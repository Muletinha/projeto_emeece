Aqui vou deixar as orientações de inicialização

- Instale e deixe o NodeJS devidamente atualizado
- XAMPP instalado rodando APACHE e MySQL (usuário ROOT sem senha)

------------backend-------------
- cd projeto_emmece/backend
- python -m venv .venv
- source .venv/Scripts/activate
- pip install -r requirements.txt
---isso vai startar o back---
- python app.py 

------------frontend-------------
- cd projeto_emeece/frontend
- npm install
- npx ng serve --project frontend

------------Up no Git------------
- git init
- git branch -M main
- git status
- git add .
- git commit -m "comentario do commit"
- git remote add origin https://github.com/Muletinha/projeto_emeece
- git push -u origin main
